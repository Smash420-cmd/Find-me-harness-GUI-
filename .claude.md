## ROOT AXIOMS (inherited everywhere, never overridden)

- **AI proposes, the engine disposes.**
- **The seam is law:** `src/engine/` is domain-free; `src/chassis/` owns all
  domain rules. `engine/` importing from `chassis/` is the cardinal violation.
- **Proof-shot or no card. Liveness always re-verifies.** (Cache holds *facts*,
  never availability.)
- **Every LLM output passes Zod before the command path.**
- **Verification core changes by human PR only.** (`src/engine/verify/`.)
- **Projection consistency:** the engine's job ends the moment it emits a
  `VerifiedResult`. Getting it on screen is the wrapper's concern. The engine
  never reaches into the UI; it emits, the projection layer renders.
